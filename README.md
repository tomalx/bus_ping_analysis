## Reporting tools and other data products to display and visualise bus delay data

This repository contains code and documentation for a set of reporting
tools and data products designed to display and visualize bus delay
data. The tools are primarily built using R, quarto and Shiny, allowing
users to interact with the data and gain insights into bus delay
patterns in the West of England.

### Data Sources

The primary source of data originates DfT bus open data portal:

##### [DfT bus data](https://data.bus-data.dft.gov.uk/)

A login is required to access the API service:

##### [bus open data API login](https://data.bus-data.dft.gov.uk/account/login/?next=/api/)
